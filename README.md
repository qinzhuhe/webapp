### 轮播图思路

- 在首尾添加图片

  1. 在开始位置添加原始的最后一张图片
  2. 在结尾位置添加原始的第一张图片

- 修改页面结构

- 修改对应的样式 

  ```css
  /* 假设是显示八张图片的轮播图 */
  .轮播图盒子 {
      width: 1000%;
  }
  .图片盒子 {
      width: 10%;
      float: left;
  }
  ```

- 设置默认的偏移：默认应该显示索引1的图片

  ```css
  transform: trnslateX(-10%); /* 参照自身 */
  left: -100%; /* 参照父容器 */
  使用定位实现的细节：当前元素必须使用 relative 定位，否则父容器无法获取正确的高度
  ```
